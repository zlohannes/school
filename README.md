# 学校论坛网站

这是一个纯前端的学校论坛网站，使用 IndexedDB 存储数据，无需后端服务器。

## 功能特性

- ✅ 纯前端，无需服务器
- ✅ 使用 IndexedDB 本地存储
- ✅ 发帖、回复、分类筛选
- ✅ 完全免费，无需任何费用
- ✅ 响应式设计，适配手机/电脑

## 部署到 GitHub Pages

1. 创建 GitHub 仓库 `school-forum`
2. 将所有文件上传到仓库
3. 启用 GitHub Pages（Settings → Pages → Source: main branch）
4. 访问地址：`https://你的用户名.github.io/school-forum`

## 本地开发

直接双击 `index.html` 即可在浏览器中运行。

## 数据库说明

数据存储在浏览器的 IndexedDB 中，不同浏览器和设备之间的数据不共享。
